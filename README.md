# Reverse Polish notation (RPN)

>_Reverse Polish notation (RPN) is a mathematical notation in which every operator follows all of its operands, in contrast to Polish notation (PN), which puts the operator before its operands. It is also known as postfix notation. It does not need any parentheses as long as each operator has a fixed number of operands. The description "Polish" refers to the nationality of logician Jan Åukasiewicz, who invented (prefix) Polish notation in the 1920s._

(Wiki 26.02.2017 -> https://en.wikipedia.org/wiki/Reverse_Polish_notation)
